Poole has basic blog support. If an input page's file name has a structure like page-title.YYYY-MM-DD.post-title.md, e.g. blog.201010-02-27.read_this.md, Poole recognizes the date and post title and sets them as attributes of the page. These attributes can then be used to generate a list of blog posts:

  • single user python imports - March 06, 2012
  • blog - March 02, 2012
  • syncing - February 24, 2012
  • links - January 01, 2001

Have a look into input/blog.md to see how it works. Feel free to adjust it to your needs.
